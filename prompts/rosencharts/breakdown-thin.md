# breakdown-thin

**Use when:** Same as breakdown, thinner bars.

**Schema:** `{ key, value, color? }` — same as breakdown.

**Example:**
```json
[
  { "key": "A", "value": 40 },
  { "key": "B", "value": 60 }
]
```

## Customization

**Colors:** Add `color` per item. Format: HEX (e.g. `"#6C5DD3"`). Rosencharts only.

**Other props:** Same as breakdown.
