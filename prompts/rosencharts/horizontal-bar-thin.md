# horizontal-bar-thin

**Use when:** Many categories, compact view (e.g. country rankings, long lists).

**Schema:** `{ key, value, color? }` — same as horizontal-bar.

**Example:**
```json
[
  { "key": "France", "value": 38.1 },
  { "key": "Spain", "value": 25.3 }
]
```

## Customization

**Colors:** Same as horizontal-bar — `color` per item, Tailwind or hex.

**Other props:** Same as horizontal-bar.
